# Open and use Assistant

Open Assistant in the Unity Editor and interact with it using natural-language prompts.

Assistant works in a conversation-based interface. You can ask questions, request suggestions, or instruct Assistant to perform actions in your project.

## Start a conversation

To begin using Assistant:

1. Open the **Assistant** window.
2. Select a [mode](xref:assistant-modes):
   - **Ask** mode provides guidance and explanations.
   - **Plan** mode generates a step-by-step plan for complex tasks.
   - **Agent** mode performs actions in your project with your approval.
3. (Optional) Attach relevant project data using the **+** button.
4. Enter your question or instruction in the text field.
5. Submit the prompt.

   Assistant displays responses in the conversation area.

## Review tool activity and permissions

When Assistant performs actions in the **Agent** mode:

- A [permission](xref:preferences#configure-permission-levels) prompt appears before changes are applied.
- [Tools](xref:assistant-tools) activity entries appear in the conversation.
- You can expand tool entries to review parameters and results.

Assistant creates or modifies assets only after you grant approval.

## Discover available capabilities

You can explore what Assistant can do by asking questions such as:

- `What skills are available in this project?`
- `What can you help me with?`

For more information about modes, refer to [Use Ask mode, Plan mode, and Agent mode](xref:assistant-modes).
